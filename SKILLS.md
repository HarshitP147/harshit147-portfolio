# SKILLS.md

This file lists the agent skills available for this codebase and how to install more.

## Installed Skills
- `skill-installer`: Install Codex skills from a curated list or a GitHub repo path.
- `skill-creator`: Create or update a skill to extend Codex capabilities.
- `find-skills`: Discover and install skills when the user asks for new capabilities.
- `apollo-client`: Guide for building React applications with Apollo Client 4.x.
- `apollo-mcp-server`: Guide for using Apollo MCP Server to connect AI agents with GraphQL APIs.
- `hashnode-api`: Hashnode GraphQL API documentation for blog data.
- `next-best-practices`: Next.js best practices for App Router, data patterns, and optimization.
- `vercel-react-best-practices`: React/Next.js performance optimization guidelines.
- `shadcn`: Manage shadcn/ui components and registries.
- `tailwind-design-system`: Build scalable design systems with Tailwind CSS v4.
- `frontend-design`: Produce distinctive, production-grade UI/UX designs.
- `ui-ux-pro-max`: UI/UX design intelligence and patterns across stacks.
- `vercel-deploy`: Deploy to Vercel.

## Installing More Skills
- Use `skill-installer` to add new skills from the curated list or a GitHub repo path.
- Use `find-skills` when you need discovery help for what skill might exist.
- Use `skill-creator` to build a custom skill if no existing skill fits.
