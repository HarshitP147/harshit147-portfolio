## Runtime and Tooling
- Package manager: `npm`.
- Dev server: `npm run dev`.
- Build: `npm run build`.
- Production run: `npm run start`.
- Lint: `npm run lint`.
- GraphQL Codegen: `npm run codegen` (config in `codegen.ts`, output `src/lib/graphql/generated.ts`).
